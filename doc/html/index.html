Le Module Image

Introduction

Le module Image regroupe trois fichier principale (main):
Le mainExemple.cpp: pour créer deux fichier d'image d'éxtension ".ppm" dans le repertoire data
Le mainTest.cpp: pour réaliser un teste de regression
Le dernier dont je n'ai pas encore finit

Pour compiler

Tapez la commande "make" dans le repertoire contenant le fichier Makefie puis ça créera dans le repertoire "bin" les executable "exemple" et "test"

Pour executer

"./bin/exemple": il va créer dans le répertoire "data" les image "image1.ppm" et "image2.ppm"
"./bin/test": il va lancer une teste de regression
"./bin/affichage": il va afficher une image à travers une fenêtre SDL

executer avec Valgrind

"make v_e" pour executer le "exemple" avec valgrind
"make v_t" pour executer le "test" avec valgrind
"make v_a" pour executer le "affichage" avec valgrind
